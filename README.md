# UICI
Repositori ini berisi kumpulan kode sumber dan proyek yang dibuat untuk memenuhi tugas-tugas kuliah di Universitas Insan Cita Indonesia (UICI). Repositori ini bertujuan untuk membagikan pengetahuan dan pengalaman dalam bidang komputer dan teknologi informasi.

Isi Repositori:

-Kode sumber untuk tugas-tugas kuliah
-Proyek-proyek yang dibuat untuk memenuhi tugas kuliah
-Dokumentasi dan penjelasan tentang kode sumber dan proyek
